# Dialogs MD3 Specs

**Source:** <https://m3.material.io/components/dialogs/specs>  
**Collected:** 2026-04-30

## Summary

- Material Design 3 dialogs expose two active token sets in the live token viewer: Dialog - Basic and Dialog - Full screen.
- Basic dialogs use a 28dp extra-large container radius with level 3 elevation, while full-screen dialogs use a 0dp container radius and keep the main container at level 0.
- Shared action affordances in both variants use label-large typography and primary-colored labels and state layers.
- Basic dialog title text maps to headline-small and supporting text maps to body-medium; full-screen dialog headlines map to title-large.
- Hover, focus, and pressed state-layer opacity tokens are surfaced through `md.sys.state.*.state-layer-opacity` system tokens.

## Implementation Notes

- Treat basic dialogs as surfaced cards with a `surface-container-high` fill, 28dp corners, 24dp outer padding, and min/max width constraints of 280dp to 560dp.
- Reuse label-large typography for dialog actions in both variants, and map hover, focus, and pressed state opacities to the surfaced `md.sys.state.*.state-layer-opacity` tokens.

- Basic dialog titles and subheads align to headline-small; full-screen headers align to title-large; supporting text aligns to body-medium.
