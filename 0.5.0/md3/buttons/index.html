    Buttons MD3 Specs

    Source: https://m3.material.io/components/buttons/specs
    Collected: 2026-04-21

    Summary

    • Material Design 3 button component specifications covering five color variants (Elevated, Filled, Tonal, Outlined, Text) and five size variants (XSmall, Small, Medium, Large, XLarge).
    • Five interactive states documented: Enabled, Disabled, Hovered, Focused, and Pressed.
    • Toggle button states for Elevated, Filled, Tonal, and Outlined variants.
    • Elevation, typography, shape, and layout tokens extracted from Light theme + Default contrast context.
    • Shape morphing uses spring animations with 0.9 damping and 1400 stiffness across all sizes.

    Implementation Notes

    • Buttons support five style variants: Elevated (shadow), Filled, Tonal, Outlined, and Text-only.
    • All button sizes support optional icon placement before or after the label, using computed icon spacing based on height.
    • Shape morphing on press uses spring animation to smoothly transition from resting shape to a temporary morph value, returning on release.
    • Toggle button functionality is available for all color variants, with distinct unselected and selected container/label/icon colors.
    • Elevation values only apply to Elevated and (deprecated) Filled Hovered state; other variants use shadow colors for visual separation.
    • Typography tokens use standard Material typography scale; custom font families can override Roboto.
    • Motion spring parameters (damping 0.9, stiffness 1400) produce a physical, responsive feel appropriate for interactive components.
